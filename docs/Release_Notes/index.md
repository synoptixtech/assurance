## v0.1 - MVP - November 2025

Initial Release of Platform, with basic set of features:

- Basic Risk Elements with core risk artifacts
- Tabular risk representation
- Status attributes on elements
- Claim tree view
- Basic approval workflow
- Create a Model and Breakdown
- Intra-diagram Copy/Paste
- Basic Version Management
- Model Autolayout
- Systems page per client
- Projects and Models per System
- Login page
- User profile page
- Sub clients with local administrators
- Basic User Management
- Clone Assurance Model
- Upload and Store Files to Evidence Elements
- Create series of PNG images for each model view
- JSON export of model
- Tabular view of element types
- Super User
- FAQ/help
- Footer
- Landing page
- Notifications Centre
- Session management
- Email OTP
- Email notifications
- Audit Log page
- All gross items tracked and logged.
